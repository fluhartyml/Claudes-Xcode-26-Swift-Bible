Appendix B: Keyboard Shortcuts Quick Reference

Claude's Xcode 26 Swift Bible

Building & Running

Cmd+R — Build and Run

Cmd+B — Build only

Cmd+. — Stop running app

Cmd+U — Run tests

Cmd+Shift+K — Clean build folder

Navigation

Cmd+Shift+O — Open Quickly (jump to any file or symbol)

Cmd+Shift+J — Reveal current file in project navigator

Cmd+Ctrl+Up/Down — Switch between .swift counterparts

Cmd+0 — Toggle navigator sidebar (left)

Cmd+Option+0 — Toggle inspector sidebar (right)

Cmd+Shift+Y — Toggle debug area (bottom)

Shift+Cmd+2 — Devices and Simulators

Editing

Cmd+/ — Toggle comment on selected lines

Cmd+[ — Indent left

Cmd+] — Indent right

Ctrl+I — Re-indent selected code

Cmd+Ctrl+E — Rename variable in scope

Cmd+D — Duplicate line

Option+Cmd+[ — Move line up

Option+Cmd+] — Move line down

Find & Replace

Cmd+F — Find in file

Cmd+Shift+F — Find in project

Cmd+G — Find next

Cmd+Shift+G — Find previous

Cmd+Option+F — Find and replace in file

Debugging

Cmd+\ — Toggle breakpoint on current line

Cmd+Y — Enable/disable all breakpoints

F6 — Step over

F7 — Step into

F8 — Step out

Cmd+Ctrl+Y — Continue execution

Interface Builder / SwiftUI Previews

Cmd+Shift+L — Library (insert views, modifiers, images)

Cmd+Option+P — Resume/refresh SwiftUI preview

Cmd+Option+Enter — Toggle preview canvas

App Store Submission

Cmd+B — Build (verify before archive)

Product > Archive — Create release archive (no shortcut by default)

Tip: Customize shortcuts in Xcode > Settings > Key Bindings.
